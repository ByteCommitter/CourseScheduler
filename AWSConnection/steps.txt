    - Convert .ppk key for aws cmd login, to act as pubKey
Connect to aws cmd using the cmd in the screenshot.

    - ssh -i path\to\your\key.ecdsa ec2-user@ec2-13-61-24-87.eu-north-1.compute.amazonaws.com
    - Set up the env also:
        -sudo dnf update -y
        -sudo dnf install java-17-amazon-corretto -y

Setup the files required for the app to run:
    - jar files from:
        -quarkus-app/lib/ # Contains all dependency JARs
        -quarkus-app/app/ # Contains your application JAR
        -quarkus-app/quarkus/ # Contains Quarkus runtime files
        -quarkus-app/quarkus-run.jar # The main executable JAR
