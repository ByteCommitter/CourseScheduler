1. Create deployment directory locally:
mkdir deployment
cd deployment

2. Essential files to transfer:
- Dockerfile
- target/quarkus-app/ (entire directory)
- src/main/resources/application.properties

3. WinSCP Transfer Steps:
- Connect using your .pem key (convert to .ppk first)
- Create directory on EC2: /home/ec2-user/timetabling
- Transfer files maintaining structure:
  /home/ec2-user/timetabling/
    ├── Dockerfile
    └── target/
        └── quarkus-app/
            ├── lib/
            ├── app/
            ├── quarkus/
            └── quarkus-run.jar

4. Post-transfer commands on EC2:
cd /home/ec2-user/timetabling
sudo docker build -t timetabling .
sudo docker run -d -p 8080:8080 timetabling

5. Access application:
http://[EC2-PUBLIC-IP]:8080
http://[EC2-PUBLIC-IP]:8080/q/swagger-ui
